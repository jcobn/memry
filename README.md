# memry

an obsidian note-based srs plugin

## installation

go to the [latest release](https://github.com/jcobn/memry/releases/latest) and download `main.js`, `manifest.json`, and `styles.css`

create the plugin folder (`<VAULT_FOLDER>/.obsidian/plugins/memry/`) and paste the files in there

enable the plugin in obsidian
